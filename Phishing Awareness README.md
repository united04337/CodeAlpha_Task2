# Phishing Awareness Training Module

## 🎯 Project Overview
An interactive, comprehensive web-based training module designed to educate users about phishing attacks, recognition techniques, and prevention best practices.

## ✨ Features
- **6 Interactive Sections:** Introduction, Recognition, Social Engineering, Real Examples, Best Practices, Quiz
- **15-Question Quiz:** Test knowledge with instant feedback
- **Real-World Examples:** Annotated phishing emails with red flag analysis
- **Responsive Design:** Works on desktop, tablet, and mobile
- **Self-Contained:** Single HTML file, no dependencies


## 🎓 Learning Objectives
- Understand what phishing is and why it works
- Recognize common phishing tactics and red flags
- Identify social engineering techniques
- Apply best practices to prevent attacks
- Respond appropriately to suspected phishing

## 🛠️ Technical Details
- **Technology:** HTML5, CSS3, JavaScript
- **Browser Support:** Chrome, Firefox, Safari, Edge
- **Dependencies:** None (fully self-contained)
- **File Size:** ~50KB
